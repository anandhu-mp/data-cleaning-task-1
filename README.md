# Employee Attrition Data Cleaning

## Project Overview
This project demonstrates data cleaning and preprocessing using the Employee Attrition dataset in Microsoft Excel.

## Objective
- Check for missing values
- Check for duplicate records
- Standardize column names
- Verify data types
- Prepare the dataset for analysis

## Tools Used
- Microsoft Excel

## Data Cleaning Process
- Opened the Employee Attrition dataset.
- Checked for missing values.
- Checked for duplicate records.
- Reviewed column names for consistency.
- Verified data types.
- Saved the cleaned dataset.
